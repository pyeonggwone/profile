# Naver Cloud Bare Metal SQL Server 라이선스 견적 포털 작업 가이드

## 1. Volume Licensing Service Center (VLSC)
Portal URL: https://www.microsoft.com/Licensing/servicecenter/LicenseAgreements/MyAgreements.aspx
Docs URL: https://learn.microsoft.com/en-us/licensing/end-customer-vlsc

### Open Value 계약 및 Software Assurance 상태 조회
1. VLSC > Licenses > License Summary 화면에서 고객 Agreement 행을 펼친다.
   - Agreement Number: `V8923741`
   - Agreement Type: `Open Value`
   - Customer Name: `한라테크 주식회사`
2. License Summary > SQL Server 제품 라인을 펼쳐 SA Coverage 정보를 조회한다.
   - Product: `SQL Server 2022 Standard Core`
   - License Quantity: `8`
   - SA Coverage Start: `2025-04-01`
   - SA Coverage End: `2027-03-31`
3. Licenses > Relationship Summary 화면에서 SA 갱신 시점을 조회한다.
   - Anniversary Date: `2027-04-01`
   - SA Renewal Window: `90 days`

### License Mobility through Software Assurance Verification 제출
1. VLSC > Downloads and Keys > Software Assurance Benefits 화면에서 License Mobility Verification Form 양식을 다운로드한다.
2. 다운로드한 양식에 호스팅 정보를 입력한다.
   - Authorized Mobility Partner: `네이버클라우드 주식회사`
   - Hosting Region: `KR-Pangyo IDC`
   - Hosting Type: `Dedicated Bare Metal Server`
   - Product: `SQL Server 2022 Standard Core`
   - License Quantity: `8`
3. VLSC > Software Assurance > License Mobility 화면에 작성한 양식을 업로드한다.
   - Submission Reference: `LM-2026-HALLATECH-01`
   - Effective Date: `2026-06-01`

## 2. Microsoft Partner Center
Portal URL: https://partner.microsoft.com/dashboard/commerce/pricing-and-offers
Docs URL: https://learn.microsoft.com/en-us/partner-center/new-commerce-experience-overview

### CSP Perpetual SQL Server 카탈로그 조회
1. Partner Center > Pricing and offers > Perpetual Software 화면을 연다.
   - Catalog Type: `Perpetual Software`
   - Product Family: `SQL Server`
   - Edition: `Standard`
   - License Type: `Per-Core`
2. SQL Server Standard Core 행을 선택해 SKU 상세 패널을 펼친다.
   - SKU ID: `DG7GMGF0M7XW:0002`
   - Channel: `CSP`
   - Term: `Perpetual`
   - Software Assurance: `Not included`

### CSP Perpetual SQL Server 견적 작성
1. Partner Center > Customers 화면에서 대상 고객을 선택한다.
   - Customer Name: `한라테크 주식회사`
   - Tenant ID: `7a3c2f1e-8d4b-4e9a-9f6c-2b1d5e8a7c3f`
   - MPN ID: `4528193`
2. 고객 상세 페이지에서 New Subscription > Perpetual Software 흐름을 연다.
   - Subscription Type: `Perpetual Software`
   - Billing Account: `한라테크-CSP-Direct`
3. Order Line에 SQL Server Standard Core 라이선스를 추가한다.
   - SKU: `SQL Server 2022 Standard Core`
   - Unit: `2-core pack`
   - Pack Count: `4`
   - Total Cores: `8`
4. Configuration 화면에 배포 메타데이터를 입력한다.
   - Deployment Target: `Naver Cloud Bare Metal Server`
   - Server Hostname: `hallatech-sqlbm-prd-01`
   - Region: `KR-Pangyo IDC`
   - Hardware Type: `Dedicated single-tenant`
5. Review 화면에서 Save as Draft 버튼을 클릭한다.
   - Quote Name: `Hallatech-SQLStd-Core-2026Q2`
   - Estimated Effective Date: `2026-06-01`
   - Status: `Draft`

### Software Assurance Add-on 견적 라인 추가
1. Pricing and offers > Software Subscriptions 화면에서 SQL Server Software Assurance를 검색한다.
   - Product Family: `SQL Server`
   - Term: `36 months`
   - Billing Frequency: `Annual`
2. 기존 Draft 견적에 Software Assurance 라인을 추가한다.
   - SKU: `SQL Server 2022 Standard Core SA`
   - Unit: `2-core pack`
   - Pack Count: `4`
   - Total Cores: `8`
3. 견적 라인 합산 결과를 Save as Draft로 저장한다.
   - Combined Quote Name: `Hallatech-SQLStd-Core-2026Q2-with-SA`
   - Status: `Draft`
