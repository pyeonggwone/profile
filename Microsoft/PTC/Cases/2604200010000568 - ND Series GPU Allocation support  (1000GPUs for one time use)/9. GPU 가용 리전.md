# GPU 가용 리전 정리

## 출처

- 원본: Azure Product Availability by Region
- 제품 영역: Virtual Machines
- 확인 기준: All geographies, Generally Available / In Public Preview / Retiring 포함
- 원본 안내: Azure는 70개 이상의 announced regions를 제공하며, geography별 제품 가용성을 확인할 수 있음

## 주의 사항

- 원본 표에서 복사된 내용에는 리전별 가용성 표시 값이 포함되어 있지 않고, SKU 행과 리전 헤더만 남아 있음.
- 따라서 아래 내용은 GPU 관련 Virtual Machines SKU와 확인 대상 리전을 읽기 좋게 정리한 것이며, 특정 SKU의 실제 가용 리전 확정 목록은 Azure Product Availability 페이지 또는 Azure Portal SKU 조회로 재확인 필요.
- `*` 표시는 Reserved Access Region으로, 특정 고객 시나리오 지원을 위해 접근이 제한될 수 있음.
- `**` 표시는 Early Access 고객에게만 제공되는 리전임.

## 1000 GPU 요청 관점의 우선 확인 대상

대규모 GPU capacity 요청에서는 아래 SKU를 우선 확인하는 것이 적절함.

| 우선순위 | SKU | 확인 포인트 |
|---|---|---|
| 1 | NDsr H100 v5-Series | 대규모 AI/HPC 학습 워크로드 후보. 리전별 quota와 capacity reservation 가능 여부 확인 필요 |
| 2 | NDsr H200 v5-Series | H200 기반 최신 SKU. preview/early access 여부 및 실제 available region 확인 필요 |
| 3 | NDsr MI300X v5-Series | AMD MI300X 기반 대규모 AI 워크로드 후보. 고객 워크로드 호환성 확인 필요 |
| 4 | NDamsr A100 v4-Series / NDasr A100 v4-Series | A100 기반 대안. H100/H200 capacity 부족 시 fallback 후보 |
| 5 | NCadsH100_v5-series / NCCadsH100_v5-series | ND가 아닌 NC 계열 H100 후보. VM size, interconnect, cluster 요구사항 적합성 확인 필요 |

## 다음 확인 방법

1. Azure Product Availability by Region에서 위 GPU SKU별 가용 리전 확인
2. Azure Portal 또는 CLI에서 subscription별 VM SKU availability 확인
3. 1000 GPU 단위 요청은 일반 quota 상향만으로 처리하기 어려울 수 있으므로, target region과 preferred SKU를 좁힌 뒤 capacity reservation 또는 별도 capacity engagement 필요 여부 확인
