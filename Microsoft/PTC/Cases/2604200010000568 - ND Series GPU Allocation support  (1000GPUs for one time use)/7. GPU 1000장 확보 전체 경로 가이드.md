## [선행 필수] 클루커스 → Azure Support 티켓 제출 (SR# 발급)

**실행 주체:** 클루커스 (Acryl AI 구독 Contributor 이상 권한 필요)

> UAT는 SR#이 없으면 Capacity 팀이 처리 불가. 반드시 이 단계를 먼저 완료할 것.

```
https://aka.ms/getazuresupport
```

또는:
```

- 티켓 제출 완료 시 **Support Request 번호(SR#)** 자동 발급

처리 시간: 3~7 영업일 (1,000장 규모는 수동 검토 가능성 높음)

## 신규 Work Item 생성

1. 위 URL 접속
2. 좌측 메뉴 → `Boards` → `Work Items`
3. 우측 상단 `+ New Work Item` 클릭 → `ACTIONS` 선택

## 주의사항

> **SR# 없으면 UAT 처리 불가.**
> 반드시 6번 파일 경로 3(Azure Support 티켓)을 먼저 제출하고 SR# 발급 후 이 UAT를 생성할 것.

> **UAT는 Microsoft 내부 직원만 생성 가능.**
> 클루커스(고객)는 직접 생성 불가 — PTC 김평권 또는 TPD 김민지가 생성.
