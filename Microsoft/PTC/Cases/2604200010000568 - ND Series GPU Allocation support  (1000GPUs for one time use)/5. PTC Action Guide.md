# PTC Action Guide — ND Series GPU 확보 내부 지원 경로

> **케이스:** ND Series GPU Allocation support (1000GPUs for one time use)
> **고객:** Acryl AI / **CSP:** 클루커스
> **작성:** PTC 김평권

---

## 개요

PTC는 고객 구독에 대한 직접 접근 권한이 없으므로 Azure Support 티켓이나 쿼터 신청을 **대신 제출할 수 없다.**
단, 아래 3가지 경로를 통해 **실질적인 지원**이 가능하다.

---

## 1. 내부 에스컬레이션 — STU/AE/PDM 연결 (가장 빠른 경로)

### 방법

1. **TPD 김민지에게 요청** → 고객사 담당 STU(Sales Technology Unit) 또는 AE(Account Executive) 연결
2. PTC가 정리한 케이스 맥락 자료(`4. Email.md`)를 STU/AE에 전달
3. STU/AE가 Microsoft 내부 Compute Capacity 채널로 pre-approval 요청 제출

### 전달할 핵심 정보

| 항목 | 값 |
|---|---|
| 고객 | Acryl AI |
| CSP | 클루커스 |
| VM SKU | `Standard_ND96asr_v4` |
| 수량 | 32 VMs (256 GPUs, 3,072 vCPU) |
| 1순위 리전 | Canada Central (2026-03 사용 이력) |
| 사용 기간 | 1주 ~ 1개월 |
| 예상 비용 | $148K (1주) ~ $635K (1개월) |
| PO 상태 | $600K 확정 완료 |

### 기대 효과

- AE/STU 경로는 Support 티켓보다 내부 우선순위가 높음
- 확정 딜($600K PO)임을 강조하면 처리 속도 향상 가능

---

## 2. UAT(Unified Action Tracker) 제출 지원

### 적용 조건

- Azure Support 티켓으로도 해결이 안 될 경우
- 대규모 capacity 확보가 필요하여 내부 weekly triage가 필요한 경우

### 방법

- **제출 가능 주체:** Microsoft 내부 담당자 (TPD 김민지, CSAM, STU 중 한 명)
- PTC는 제출용 요약 자료 작성 지원 가능

### 주의사항

> ND Series VM 용량 요청이므로 **Azure OpenAI UAT(aicapacity@microsoft.com)가 아닌**  
> **Compute Capacity 팀 채널**로 제출해야 한다.

### UAT 제출 시 포함할 내용

```
Customer: Acryl AI (CSP: Cloukers)
VM SKU: Standard_ND96asr_v4
Count: 32 VMs (256 GPUs)
Region: Canada Central (primary), East US / West US 2 (fallback)
Duration: 1–4 weeks
Business impact: $600K committed PO, GPU availability is sole blocker
Prior history: Same SKU deployed in Canada Central, March 2026
```

## 3. Justification 문서 대신 작성 → 클루커스 전달

### 역할 분담

| 주체 | 역할 |
|---|---|
| **PTC** | Justification 텍스트 작성 완료 (`4. Email.md` 내 포함) |
| **클루커스** | Azure Portal에서 복사 붙여넣기 후 제출 |

```

HPC cluster. Customer: Acryl AI / CSP: Cloukers.

Use case: Distributed multi-GPU orchestration validation for "Jonathan" MLOps platform.
Customer previously deployed identical SKU in Canada Central in March 2026
via Azure Support ticket.

```

## 권장 실행 순서

```
[PTC] 4. Email.md 전달 → TPD 김민지
    └─ [TPD 김민지] STU/AE 연결 요청 (Action 1)
    └─ [클루커스] Azure Portal 쿼터 신청 (Action 3 - Justification 복사)
         └─ 자동 승인 실패 시
              └─ [클루커스] Azure Support Capacity Request 티켓 (Action 3)
              └─ [TPD/CSAM] UAT 제출 (Action 2) ← 병행
```
