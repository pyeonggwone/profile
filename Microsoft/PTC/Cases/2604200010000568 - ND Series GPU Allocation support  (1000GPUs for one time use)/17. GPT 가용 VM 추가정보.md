# 추가 정보 기준 가용 VM 리스트

## 산정 기준

`14. GPU 추가정보.md`의 고객 요구사항과 `10. VM별상세스펙.json`의 GPU 수량을 기준으로 아래처럼 산정했습니다.

- 1순위 요청안: 총 256 GPUs = 고스펙 GPU 128장 + 저스펙 GPU 128장
- 2순위 fallback: 총 128 GPUs = 고스펙 GPU 64장 + 저스펙 GPU 64장
- 상시 사용 저스펙 GPU: 16 GPUs
- 고스펙 GPU는 RDMA/InfiniBand 지원이 필수이므로 `NDsr H100 v5-Series`, `NDsr H200 v5-Series`만 기준으로 사용
- VM 개수는 각 VM series에서 사용 가능한 최대 GPU 수량의 SKU 기준으로 계산
- 동일 Azure region 안에서 고스펙 RDMA VM과 저스펙 GPU VM을 함께 확보할 수 있는 조합만 가용 후보로 정리

## 케이스 구분 기준

- 케이스 1: 고스펙 RDMA VM과 저스펙 RDMA VM을 같은 region에서 함께 구성하는 안
- 케이스 2: 고스펙 RDMA VM과 저스펙 일반 GPU VM을 같은 region에서 함께 구성하는 안
- 케이스 3: 고스펙 RDMA VM 기준으로 저스펙 RDMA VM과 저스펙 일반 GPU VM을 모두 선택지로 둘 수 있는 안
