# MSX Opportunity → UAT 전체 흐름 (처음부터 끝까지)

## [0] 시스템 시작

접속:
https://microsoftsales.crm.dynamics.com/main.aspx

경로:
Apps → Sales Hub

참고:
https://learn.microsoft.com/en-us/dynamics365/sales/intro-saleshub [1](https://learn.microsoft.com/en-us/dynamics365/sales/intro-saleshub)

---

## [1] 고객(Account) 확인

경로:
Sales Hub site map → Accounts 또는 Customer 360 → 고객 검색

목적:
- 고객(Account) 존재 확인
- 기존 Opportunity 확인

---

## [2] Opportunity 확인 또는 생성

경로:
Sales Hub site map → Opportunities

문서 기준:
“Sales Hub site map → Opportunities → New” 

### [2-1] 기존 Opportunity 있음
- 해당 Opportunity 사용

### [2-2] 기존 Opportunity 없음
경로:
Sales Hub site map → Opportunities → New

---

## ✅ 현재 상태 (BLOCKED)

현재 단계:
[2] Opportunity 생성 단계

현재 상태:
- Opportunities 목록 접근 가능
- New 버튼 없음

원인:
- Opportunity 생성 권한 없음

필요 권한:
MSX Dynamics 365
→ Opportunities: Read/Write
→ (보통 Managed Account Seller)

참고:
권한 없으면 Opportunity 생성 UI 자체가 안 보임 

---

## [3] Milestone 생성

경로:
Opportunity 상세 → Milestones 탭 → Add Milestone

---

## [4] Milestone 값 설정 (AI Infra GPU 요청용)

필수 값:
Milestone Workload = AI: AI Infra (GPUs)
Milestone Status = Blocked
Milestone Status Reason = Capacity/Service Availability

---

## [5] UAT 생성

경로:
Milestone → Add UAT Action

---

## [6] UAT 제출

- UAT Form 입력
- GPU Capacity 요청 입력
- Submit

---

## [7] 완료 상태

- UAT 생성 완료
- AI Capacity Triage 전달

---

# 핵심 요약

전체 흐름:
Account → Opportunity → Milestone → UAT

현재 위치:
Opportunity 생성 단계에서 권한 부족으로 진행 불가

필요 조치:
- Managed Account Seller 승인
또는
- 권한 있는 사용자에게 Opportunity+Milestone 생성 요청
