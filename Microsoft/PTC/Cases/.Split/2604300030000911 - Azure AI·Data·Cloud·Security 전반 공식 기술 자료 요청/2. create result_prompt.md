---
description: note_templete.jsonl을 참조하여 note.md를 완성한다.
---

[Input]

- `1. problem list.md`
- `jsonl/note_templete.jsonl`
- `reference/case_statement.md`

[Task]

`1. problem list.md`의 각 제목을 하나의 검토 주제로 보고, 제목별로 `2_note_{제목}.md` 파일을 만든다.

작성 전에 `reference/case_statement.md` 로컬 자료는 사실 확인용으로만 사용하고, 최종 파일에는 Microsoft 공식 문서, Azure Portal, Microsoft 내부 링크처럼 URL이 있는 출처만 원문 URL 그대로 남긴다.
내부 저장소의 디렉토리 이름을 출처로 사용하면 안된다.

```text
[Customer_Statement]
[pre_scoping]
[scoping]
[research]
```
[Output]
- 문제 목록의 제목 순서대로 `2_note_{제목}.md` 생성
