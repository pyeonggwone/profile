# PPT 자료 복사 결과

고객 요청의 9개 자료 유형에 맞춰 각 카테고리별로 가장 가까운 PPT를 1개씩 선택해 복사하였다.

## 선택 자료

### 1. 제품 소개 자료

**Agent Builder in Microsoft 365 Copilot customer deck_KO.PPTX**

Microsoft 365 Copilot의 Agent Builder를 고객 관점에서 설명하는 자료다. 제품 개요와 활용 시나리오를 소개하기에 적합하다.

### 2. 기술 자료

**Foundry Agent Service L300 Pitch Deck_KO.PPTX**

Foundry Agent Service를 L300 수준에서 다루는 자료다. 서비스 구조와 기술 구성 방식을 설명하는 데 적합하다.

### 3. 개발 가이드

**Playbook - MCP Apps in Copilot Agents Overview_KO.PPTX**

MCP Apps와 Copilot Agents 연결을 다루는 자료다. Agent 구성과 앱 연동 개발 흐름을 참고하기 좋다.

### 4. 아키텍처 자료

**Container Migration Solution Accelerator Overview_KO.PPTX**

Container Migration의 솔루션 구조와 적용 방식을 설명하는 자료다. Cloud 아키텍처 검토용으로 활용할 수 있다.

### 5. PoC 및 Demo 자료

**Copilot Studio EBC (in-person)_KO.PPTX**

Copilot Studio를 고객 briefing과 demo 맥락에서 설명하는 자료다. PoC 주제 선정과 demo 흐름 기획에 적합하다.

### 6. 운영 및 확장 가이드

**Cloud Accelerate Factory - Service Descriptions_KO.PPTX**

서비스 범위와 제공 항목을 설명하는 자료다. 도입 이후 운영 및 확장 논의에 활용하기 좋다.

### 7. Change Management 및 Adoption 자료

**Pitch Deck - Change Management L200_KO.PPTX**

Change Management를 L200 수준에서 설명하는 자료다. 조직 도입, 사용자 확산, 내부 전파 계획에 적합하다.

### 8. 보안 및 거버넌스 자료

**Frontier CoE - AI Governance & Security_KO.PPTX**

AI Governance와 Security를 직접 다루는 자료다. 보안 검토와 거버넌스 논의에 활용하기 좋다.

### 9. CSP 및 파트너 비즈니스 자료

**Session 1-CSP Overview_KO.pptx**

CSP 사업과 프로그램 개요를 설명하는 자료다. 파트너 비즈니스 소개 자료로 활용하기 적합하다.

## 복사 파일 수

총 9개 파일
