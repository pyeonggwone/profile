# PDF 텍스트 기능 통합 비교

기준: 공식 문서/API/CLI에서 확인되는 기능만 적었다. `지원`은 해당 기능을 공식 API로 직접 제공하는 경우, `부분`은 저수준 접근이나 보조 정보는 제공하지만 최종 기능은 직접 구현해야 하는 경우, `미지원`은 공식 기능으로 보기 어려운 경우다.

## 지원 합계

아래 합계는 `## 전체 기능 테이블`의 32개 기능 행을 기준으로 계산했다. `지원 합계`는 `지원 + 부분`이다.

| 라이브러리 | 지원 | 부분 | 미지원 | 지원 합계 |
