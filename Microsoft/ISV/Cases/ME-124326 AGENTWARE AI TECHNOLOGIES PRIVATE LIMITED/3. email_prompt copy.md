End each section with `**Reference:**` and include only official Microsoft links when available.

Official Microsoft domains only: `learn.microsoft.com`, `azure.microsoft.com`, `partner.microsoft.com`, `marketplace.microsoft.com`, `github.com/Azure`, `github.com/microsoft`.

`## Next checkpoints`: include only meeting-specific next actions related to the follow-up topics.
